
# 角色扮演工具 (Cosplay Tool)

你可以cosplay任何你想成为的角色。

---

## 版本选择

| 条件 | 推荐使用 |  |
| :--- | :--- | ---- |
| 拥有 Gemini API Key | **TypeScript 版本** | npm install；npm run dev |
| 拥有 Gemini vertex_credentials.json | **Python 版本** | pip install -r requirements.txt |

---
